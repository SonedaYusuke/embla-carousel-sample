# Embla Carousel Sample

こちらはアドベントカレンダー2025 12/21の記事で使用しているEmbla Carouselのサンプリリポジトリです。


## Setup

Make sure to install dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
